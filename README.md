## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- [x] Install all dependencies (using the `npm install` command).
- [x] Run the development web server using the `node express_server.js` command.
- [x][x][x]Enjoy!
